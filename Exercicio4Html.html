Exercício 4 HTML

Estrutura de um Documento HTML

A estrutura de um documento html é definida pela declaração de tags e de uma intrução. Essa estrutura é interpretada pelo navegador que a renderiza e então exibe para o usuário

A seguir vamos descrever cada uma dessas tags e instrução que são obrigatórias no documento html

Instrução !DOCTYPE html

A instrução !DOCTYPE html é utilizada para definir a versão do html, neste exemplo estamos usando o html 5. É importante frisar que a instrução !DOCTYPE html não é uma tag html.

Tag head

A tag head é utilizada para configurações do navegador e representa o cabeçalho do documento html. Nesta tag podemos colocar meta tags de configuração e outras referencias que serão utilizadas pelo navegador para ajustar a sua exibição. Além disso, temos também um título que aparece no topo da página.

Tag body

A tag body é utilizada para exibir toda parte vísivel do documento html. Nela podemos ter várias outras tags, que são renderizadas pelo navegador.

Tag html

A tag html é a tag raiz do documento html e contém as outras tags estruturantes que formam o documento html. Em síntese essa tag delimita a totalidade do documento html.
