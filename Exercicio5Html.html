Exercício 5 HTML - Principais elementos

Os Principais elementos html incluem tags para estrutura, texto, mídia e todo tipo de interatividade. Essas tags são interpretadas pelo navegador e utilizadas para construção do documento html. A seguir descrevemos as principais tags não estruturantes.

Tags parágrafo - p, linha horinzotal - hr e quebra de linha - br

Tag p - parágrafo

A tag parágrafo é utilizada para definir parágrafos. Neste bloco estamos utilizando a tag p que vem dentro do sinal de diamante.


Tag br

A tag br é utilizada para quebra de linha e podemos utilizar quantas quebras forem necessárias. Abaixo deste bloco estamos usando três br para pular três linhas.




Tag hr

A tag hr é utilizada para criar uma linha horinzotal que ocupa toda largura da página/documento html. Abaixo utilizamos a tag hr para separar o conteúdo deste parágrafo do próximo
