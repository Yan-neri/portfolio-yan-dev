Box Model

Em css, o termo Box Model ou modelo de caixa é usado quando se fala em design e layout web. O Box Model é essencialmente uma que envolve cada elemento html

Cada caixa consiste em, quatro partes: conteúdo, preenchimento, bordas e margens Neste exemplo podemos visualizar de forma clara a representação do Box Model
