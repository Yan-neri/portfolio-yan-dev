Seletor de ID

O seletor de ID é aquele que aplica a regra css a um componente html que tenha um atributo id no componente HTML. Esse seletor de id deve ser aplicado a um único elemento HTML, haja vista, que o id em tese deve ser único. Além disso, esse seletor terá prioridade máxima sobre os demais seletores (de classe e de tipo)

Neste exemplo estamos utilizando seletores de id para estilizar o documneto html,
