Exemplo 7 HTML - Uso das tags <ol>, <ul> e <li>

As tags <ol>, <ul> representão listas em html, e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada ja a tag <ul> representa uma lista não ordenada.

Em ambas as listas para representar os elementos da lista usamos a tag <li>

Além disso, estas listas podem conter outras lista de forma alinhadas.

Etapas para Desenvolver um Site(Lista Ordenada)

  1. Planejamento
    1. Definir o objetivo do site
    2. Identificar o público-alvo
    3. Criar um mapa do site(site map).
  2. Design
    1. Criar wireframes e prótotipos
    2. Definir uma paleta de Cores e tipografia
    3. Desenvolver o layout no Figma ou Adobe XD.
  3. Desenvolvimento
    1. Configurar o ambiente de Desenvolvimento
    2. Codificar o front-end(HTML, CSS e JavaScript)
    3. Implementar o banck-end(C#, Java, Go, PHP).
  4. Teste
    1. Testar a responsividade em diferentes dispositivos
    2. Realizar teste de usabilidade
    3. Corrigir bugs identificados.
  5. Publicação
    1. Escolher um provedor de hospedagem
    2. Configurar o domínio e SSL
    3. Fazer o deploy do site.
