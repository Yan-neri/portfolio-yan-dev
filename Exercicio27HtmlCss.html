Exemplo 27 CSS - Introdução à Propriedade CSS Display

O que é a Propriedade Display?

A propriedade display no CSS especifica o comportamento de renderização de um elemento, ou seja, como ele é exibido na página e como interage com os elementos ao seu redor.

Principais Valores de Display

  • block: O elemento é exibido como um bloco, ocupando toda a largura disponível e iniciando em uma nova linha.
  • inline: O elemento é exibido na mesma linha, ocupando apenas o espaço necessário para seu conteúdo, sem iniciar uma nova linha.
  • inline-block: Combina características dos dois anteriores; o elemento é exibido na mesma linha, mas permite definir largura e altura.
  • flex: Define o elemento como um contêiner flexível, permitindo um layout mais dinâmico dos elementos filhos.
  • grid: Define o elemento como um contêiner de grade, facilitando a criação de layouts bidimensionais complexos.
  • none: O elemento não é exibido na página e não ocupa espaço.

Display: block

Display: block: O elemento div com a classe .block é exibido como um bloco, iniciando em uma nova linha e ocupando toda a largura disponível. A cor de fundo azul claro e a margem inferior de 10 pixels ajudam a visualizar sua extensão.

Display: inline

Display: inline: Os elementos com a classe .inline são exibidos na mesma linha, um ao lado do outro, ocupando apenas o espaço necessário para seu conteúdo. A cor de fundo verde claro e a margem direita de 10 pixels destacam cada elemento.

Display: inline-block

Display: inline-block: Os elementos
com a classe .inline-block são exibidos na mesma linha, mas permitem a definição de largura (150 pixels) e altura (50 pixels). A cor de fundo coral claro, a margem de 5 pixels e o texto centralizado facilitam a visualização de suas dimensões.
